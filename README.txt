# Jogo de Xadrez no windows form
* Este é uma adaptação de um projeto de um jogo de xadrez em console escrito em C# e com auxílio do Visual Studio Community;
* Este projeto foi desenvolvido com base na proposta do curso "C# COMPLETO 2019 Programação Orientada a Objetos + Projetos", disponibilizado na Udemy e desenvolvido por Nelio Alves.
* No entanto, diferente o original, esse foi feito para funcionar com interface gráfica no windows form.
